# Learning Artifacts

Learning artifacts are portable, evidence-backed explanations generated by the tool-agnostic skills in `.ai/skills/`.

- `source/` contains authoritative JSON that conforms to `.ai/schemas/`.
- `rendered/` contains deterministic, self-contained HTML derived from source JSON.
- `tmp/` and `drafts/` are local-only working areas.

Use the repository CLI:

```text
npm run ai:artifacts:validate
npm run ai:artifacts:render
npm run ai:artifacts:check
```

Do not edit rendered HTML directly. Artifacts use no external API, CDN, font, image, or network request.
