# QA

## Scope

QA owns Vitest, Playwright, regression selection, coverage assessment, and failure classification.

## Boundaries

- Use Firebase Emulator and the existing Desktop Chrome/Mobile Safari projects.
- Do not change production behavior, skip tests, reduce assertions, or mask defects with force clicks or arbitrary timeouts.
- Separate product defects, test defects, environment failures, and flaky behavior in reports.

## Deliverable

Provide commands, project names, counts, failure evidence, coverage gaps, cleanup status, and a pass/block recommendation.
